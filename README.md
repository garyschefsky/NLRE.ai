# New Luna Ventures Website

A modern, clean website for New Luna Ventures - a real estate development company focused on revitalizing San Francisco's urban core.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js app directory with pages
- `components/` - Reusable React components
- `public/` - Static assets (images, etc.)

## Pages

- `/` - Landing page
- `/what-we-do` - Services and offerings
- `/our-team` - Team information
- `/clients-we-serve` - Client information
- `/projects` - Projects and offering memoranda
- `/connect` - Contact form

## Adding Images

Place images in the `public/` directory and reference them using `/image-name.jpg` in your components.
